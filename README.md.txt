1. Project Description
This project analyzes the Titanic dataset to understand what factors affected passenger survival. 
We used data visualization and a Logistic Regression model to predict if a passenger survived.

2. Dataset
- Source: Kaggle - Titanic Dataset
- File: `titanic-dataset.csv`
- Rows: 891 passengers
- Key Columns: PassengerId, Survived, Pclass, Name, Sex, Age, Fare, Embarked

4. How to Run
1. Install required libraries:
pandas -> Data Handling
matplotlip, seaborn -> Visualization
scikit-learn -> Machine Learning
jupyter -> Notebook/Environment

2. Open `titanic_dataset.ipynb` in Jupyter Notebook or VS Code
3. Run all cells to generate the charts and the report

 5. Key Findings
1. Females had a much higher survival rate than males.
2. Passengers in 1st Class had better chances of survival.
3. The Logistic Regression model achieved ~78% accuracy in predicting survival.
